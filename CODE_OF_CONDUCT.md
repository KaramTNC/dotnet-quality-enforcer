# Contributor Covenant Code of Conduct

## Our pledge

We are committed to making participation in this project a respectful and welcoming experience for everyone, regardless of background or identity.

## Expected behavior

Examples of behavior that contribute to a positive environment include:

- being respectful and considerate;
- discussing ideas and code without personal attacks;
- accepting constructive feedback; and
- focusing on what is best for the project and its users.

Unacceptable behavior includes harassment, discrimination, intimidation, personal attacks, and publishing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this code. Repeated or serious violations may result in a temporary or permanent ban from project spaces.

Report conduct concerns privately to the repository maintainers through GitHub. Security vulnerabilities should follow the process in [SECURITY.md](SECURITY.md).

This code of conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
